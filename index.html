Why TypeScript?

  • Eliminates many human error bugs
  • Potential bugs are caught in the editor, not at runtime
  • Makes code cleaner, easier to maintain
  • Especially effective in large codebases with many contributors
  • Easy to incrementally adopt TS in an existing JS project
  • Encourages you to write better code, think about architecture of code
  • TS is becoming standard at large companies and accross industry
